# E-Commerce Sales & Customer Analytics (MySQL)

This project analyzes customer behavior and sales performance
using MySQL for an e-commerce platform.

## Technologies Used
- MySQL 8.0
- SQL Analytics

## SQL Concepts Covered
- Joins
- Aggregate Functions
- Subqueries
- Date Functions
- Business Analytics Queries

## Use Case
Used by business teams to track revenue, customer spending,
and product performance.
